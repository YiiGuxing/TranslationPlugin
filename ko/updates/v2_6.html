버전 2.6(2019-12-05)

Translation v2.6을 사용해주셔서 감사합니다. 해당 버전에는 다음과 같은 주요 업데이트가 포함되어 있습니다:

  • 문서 주석 번역 - 원클릭으로 문서 주석 번역 가능
  • YOUDAO 번역에서 지원하는 언어 - 다양한 새로운 언어 지원
  • 글꼴 미리보기 - 실제 텍스트 적용 상황을 표시
이곳에서 해당 릴리스 노트를 확인하실 수 있습니다.

번역

문서 주석 번역

이제 코드에 있는 문서 주석에 대해 원클릭으로 번역을 할 수 있으며, 더 이상 텍스트를 수동으로 선택한 다음 번역할 필요가 없습니다. 또한 문서 주석의 일부 요소(문서 특수 기호, HTML 요소 등)에 의한 간섭 역시 걱정하지 않아도 됩니다.

참고: 이 기능을 사용할 때는 최상의 번역 결과를 위해 Google 번역을 사용하고 IDE를 최신 버전으로 업그레이드하는 것이 좋습니다.

옵션

글꼴 미리보기

실제 텍스트 적용 상황을 보여주기 위해 옵션 패널에서의 글꼴 미리보기를 최적화했습니다.이전에는 IDE 기본 글꼴로 인해 미리보기 텍스트에서 선택한 글꼴이 제대로 적용되지 않고 일부만 적용되어 미리보기와 플러그인 UI에 표시된 글꼴이 일치하지 않았었습니다.
