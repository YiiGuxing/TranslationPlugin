Version 2.5 (2019-10-20)

Welcome to the v2.5 Release Notes. There are a number of updates in this release that we hope you will enjoy. Some of the main highlights are：

  • Youdao Dictionary view - a new dictionary view
  • Labels for Glossary words - group and categorize words in the Glossary
  • Glossary import/export - share your Glossary

Imports and exports

Glossaries can now be imported or exported to allow for Glossary sharing.The Glossary can be exported in the following formats：

  • JSON (can be used for Glossary imports)
  • XML (can be used for Glossary imports)
  • Youdao XML (for importing into the Youdao dictionary, but not for Glossary imports)
