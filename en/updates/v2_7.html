Version 2.7 (2020-01-08)

Welcome to Translation v2.7, which includes the following major updates：

  • More languages supported for document comment translations - supports document comment translation in languages such as Go, Dart, Python, etc.
  • Column Selection Mode - supports translations in Column Selection Mode
If you would like to read these Release Notes, please click here.

Translation

Translation of document comments

Document comment translation now supports multiple new languages, including：

  • Go
  • Dart
  • Python
  • C, C++, Objective-C/C++

Column Selection Mode

The text selected through Column Selection Mode can now be read and translated in its entirety, rather than just the last line.
